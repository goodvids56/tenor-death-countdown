The Tenor API is dying

On June 30, 2026, Google shuts down the public Tenor GIF API. For years it quietly answered the GIF searches inside Discord, WhatsApp, X, Bluesky, Telegram, and a pile of apps that never told you they were using it.

what happened

Google announced the shutdown on January 13, 2026. New API sign-ups stopped that day. On June 30, 2026 the keys that still work go dark too, and calls to the API start returning errors instead of GIFs.

Tenor's own website and app keep running. The part that let every other app pull from its GIF library does not.
