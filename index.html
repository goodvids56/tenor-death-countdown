The Tenor API is dying

On June 30, 2026, Google shuts down the public Tenor GIF API — the service that quietly powered GIF search in Discord, WhatsApp, X, Bluesky and countless other apps.

What happened?

Google announced on January 13, 2026 that it would deprecate and discontinue the public Tenor API. New sign-ups stopped immediately, and on June 30, 2026 every existing API key stops working — requests just start returning errors. Tenor's website and app live on; the technology connecting its GIF library to the rest of the internet does not.
